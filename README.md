# Voting-Management-System
This is window based application using jframe

## Software
- Netbeans
- https://netbeans.org/downloads/start.html?platform=windows&lang=en&option=all

## Front End
- java swing
## Back End 
- Postgres data base
- https://www.postgresql.org/download/windows/
- username= postgre
- password= DBMS
